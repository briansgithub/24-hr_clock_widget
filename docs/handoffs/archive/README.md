# Archived branch handoffs

Frozen closed-form handoffs and append-only per-branch histories for branches that are no longer active.

Path mirrors:

- `docs/handoffs/feature/foo.md` → `docs/handoffs/archive/feature/foo.md`
- `docs/handoffs/feature/foo.history.md` → `docs/handoffs/archive/feature/foo.history.md`

Rules: freeze both files at archive time; later edits only for factual corrections. See [MASTER_HANDOFF.md](../MASTER_HANDOFF.md) and append one close entry to global [HISTORY.md](../HISTORY.md) when archiving.
